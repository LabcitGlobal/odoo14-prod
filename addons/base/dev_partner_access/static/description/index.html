Overview

Every business having lot of customers and its necessary sales person can see their own customer So make thing happen we have made small app in odoo in which salesperson can show their own customers and also can make a own customer's sale orders.
With help of this app, user can assign multiple sales person for one customer so it will reduce the complexity having one customer have one sales person. By allow Sales Person you can add many sales person for one customer as below screen

Feature

Own Customer Only can see only him self as Customer
Own Customer Only can not create new customer
Allow Salesperson can see to allocated Customer
All Customer can see all Customer
Assign Mass Salesperson to Multiple customer in one click
